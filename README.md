# async-macros
[![crates.io version][1]][2] [![build status][3]][4]
[![downloads][5]][6] [![docs.rs docs][7]][8]

Macros for async-std.

- [Documentation][8]
- [Crates.io][2]
- [Releases][releases]

## Installation
```sh
$ cargo add async-macros
```

## Safety
This crate uses `unsafe` for pin projections.
